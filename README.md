Hectare
=======

This is a terrain drawing program.

To get something interesting to happen, use the hectare binary like this
./hectare data/terrain.tga data/texture.tga

Controls are:

    e : move forward
    d : move back
    s : move left
    f : move right
    w : move down
    r : move up
    
    S : rotate left
    F : rotate right
    E : rotate down
    D : rotate up
